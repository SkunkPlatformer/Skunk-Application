How to Use:

Use with the !help to view More Commands.

Information:
SkunkApp provides features for server protection, utility commands, pet commands, and moderation commands. To add an app to SkunkApp, scroll down in the box, click the "Show/Hide" button to display the options, and then click the "Add App" button.

How to Use These Commands for SkunkApp:
SkunkApp includes the following commands:
Basic Commands:
!pet | View your pet.
!register-pet | Register a new pet.

Moderation Commands:
/mod-ban | Ban a member.
/mod-kick | Kick a member.
/add-filter | Create a filter to block specific words and automatically delete messages containing them. (Filtered words will be protected and may result in message deletion.)

Basic Slash Commands:
/report | Report a user or username in games.
/hackreport | Report a user using hacks in games or Discord.
/chatreport | Report a user for inappropriate language or other reasons in games or Discord.
/feedback | Send feedback to the bot developer. Use this to report issues, errors, or glitches.
/get-information | View a user's personal information in a private DM. Do not share any personal information.

That's all of the commands!
